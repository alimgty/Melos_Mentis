About the Project

This project presents an AI-powered music therapy system that detects users' emotions and environmental context in real-time to generate personalized therapeutic music. Leveraging deep learning models like VGG16 for facial emotion recognition and YOLOv5s for context detection, the system ensures that each music piece is emotionally aligned with the user's current state. It also integrates a RAG-based AI chatbot, built with Mistral 7B, to offer interactive, empathetic conversations that further enhance emotional support. The backend is built using Flask and FastAPI for modularity, while MongoDB manages user authentication and personalization. A responsive and intuitive frontend enables seamless real-time interaction. By combining music, emotion recognition, and AI-driven dialogue, this platform offers an innovative solution for mental wellness, aiming to reduce stress and improve emotional stability through dynamic, adaptive therapy sessions.
